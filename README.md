# Ocelloids Python Library

A Python library for interacting with Ocelloids.

## Requirements

- Python 3.8+
- [uv](https://docs.astral.sh/uv/)

```bash
uv install
```

## Usage

To run an example using `uv`:

```bash
uv run examples/main.py
```

## Development

To lint and format your code with `ruff`:

```bash
ruff format
```
